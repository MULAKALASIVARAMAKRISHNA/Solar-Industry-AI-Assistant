# Solar AI Assistant

## Setup
1. Clone repository
2. `pip install -r requirements.txt`
3. Create `.env` with your OpenRouter API key
4. `python app.py`

## Features
- Technical & non-technical support
- Cost analysis
- Regulatory guidance
- Market insights
